# 📊 Dashboard de Controle de Estoque

Projeto desenvolvido em Power BI para análise e monitoramento de estoque.

## Ferramentas utilizadas

- Power BI
- Power Query
- DAX

## Objetivo

Fornecer uma visão dos principais indicadores de estoque através de dashboards interativos.

## Principais recursos

- Indicadores em cartões (KPIs)
- Filtros dinâmicos
- Gráficos para análise temporal
- Tabelas detalhadas
